# TP Cloud Computing

Ce dépôt contient un petit projet réalisé dans le cadre d'un TP de cours sur le cloud computing.

## Objectif

L'objectif est de mettre en place un client Node.js connecté à Supabase pour gérer des fonctions simples d'authentification et de création de profil utilisateur.

## Technologies utilisées

- Node.js
- Supabase
- `@supabase/supabase-js`
- `dotenv`

## Structure du projet

- `client.js` : initialisation du client Supabase
- `auth.js` : fonctions d'inscription, de connexion et de déconnexion

## Variables d'environnement

Créer un fichier `.env` à la racine du projet avec les variables dans .env.sample

## Installation

```bash
npm install
```
